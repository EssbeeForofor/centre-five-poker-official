Versioning Policy

Centre Five Poker uses a simple incremental versioning system:
Major version (v2.x) — Reserved for rule changes affecting game flow or structure.
Minor version (v1.x) — Added clarifications, wording improvements, examples, or optional modes.
Patch (v1.1.x) — Typos, formatting fixes, minor corrections.
